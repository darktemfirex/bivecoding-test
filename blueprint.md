# Lotto Generator Blueprint

## Overview
A modern, high-performance Lotto number generator built with pure HTML, CSS, and JavaScript. The application focuses on a premium user experience, incorporating beautiful animations, responsive design, and modern web standards.

## Detailed Outline & Features
- **Lotto Generation Logic:**
  - Generates 6 unique random numbers from 1 to 45.
  - Automatically sorts numbers in ascending order.
- **Visual Design:**
  - **Premium UI:** Glassmorphism effects, subtle noise textures, and deep multi-layered shadows.
  - **Dynamic Colors:** Lotto balls colored based on standard ranges (1-10: Yellow, 11-20: Blue, 21-30: Red, 31-40: Gray, 41-45: Green).
  - **Animations:** Smooth entry animations for balls when generated.
  - **Responsive Layout:** Works perfectly on mobile and desktop using Container Queries and Flexbox/Grid.
- **Components:**
  - `LottoBall`: A custom Web Component for rendering individual balls with their specific styles.
  - `HistoryCard`: Displays previous generation results.
